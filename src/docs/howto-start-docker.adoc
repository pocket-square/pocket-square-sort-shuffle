= HOWTO: start service as docker container

== Cleaning previously started container

[source,shell]
----
docker ps -a
docker kill pocket_square_sort_shuffle; docker rm pocket_square_sort_shuffle
----

== Start service

[source,shell]
----
docker pull pocketsquare/pocket-square-sort-shuffle:$TAG
docker run -dit --name pocket_square_sort_shuffle --link pocket_square_articles -p 28104:5000 pocketsquare/pocket-square-sort-shuffle:$TAG
----
